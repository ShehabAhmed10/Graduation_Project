Yemen Tourism Mobile App

This folder contains the Flutter mobile app skeleton for the Yemen Tourism project.

Next steps:
- Install Flutter SDK and ensure `flutter --version` is available.
- From this folder run `flutter pub get` and open in Android Studio / VS Code.
- Implement services and providers according to `backend_contracts/api_contracts.md`.

Key folders:
- lib/config: API endpoints and constants
- lib/core: networking, theme, utils
- lib/data: models and services
- lib/providers: state management
- lib/ui: screens and widgets
